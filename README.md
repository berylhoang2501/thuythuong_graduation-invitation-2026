# Thiệp mời Lễ Tốt Nghiệp 2026

Website thiệp mời tốt nghiệp của **Hoàng Ngọc Thủy Thương**.

- Thời gian: **10:00 - 30/07/2026**
- Địa điểm: **Nhà hát Hòa Bình**
- Địa chỉ: **240 Đường 3 tháng 2, Phường Hòa Hưng, TP.HCM**

## Cá nhân hóa tên khách mời

Thêm `?to=` vào cuối URL, ví dụ:

```text
https://berylhoang2501.github.io/thuythuong.github.io/?to=Nguyen%20Van%20An
```

## Deploy bằng GitHub Actions

1. Upload toàn bộ nội dung của project này lên branch `main`.
2. Vào `Settings > Pages`.
3. Tại `Source`, chọn `GitHub Actions`.
4. Mỗi lần push lên `main`, workflow `.github/workflows/deploy.yml` sẽ tự build và deploy.

URL dự kiến:

```text
https://berylhoang2501.github.io/thuythuong.github.io/
```

## Chạy local

```bash
npm ci
npm run develop
```
